# FrankenCommander (fc)

Norton Commander-style file manager built on FrankenTUI.

## Run (once Rust is installed)

```bash
cargo run
```

Requires Rust nightly (FrankenTUI uses nightly + edition 2024).

## Keys

- `Tab` switch panes
- Arrow keys navigate
- `Enter` open directory
- `Backspace` up directory
- `F3` view file
- `F5` copy
- `F6` move
- `F7` mkdir
- `F8` delete
- `F10` quit
